This project was born out of a challenge: How can a sports franchise use historical data to build a championship-winning squad? Focusing on the Royal Challengers Bangalore (RCB), I conducted a deep-dive SQL analysis of IPL historical data. The project simulates the role of a Sports Data Analyst preparing for a mega-auction, providing the franchise with evidence-based player recruitment and match-day strategies.

Key Features & Analysis
Data Cleaning & Integrity: Standardized team names and resolved data inconsistencies (e.g., correcting legacy team names like Delhi Daredevils) to ensure reporting accuracy.

Player Performance Metrics: Calculated advanced KPIs including Strike Rates, Batting Averages, and Bowling Economies to identify "Gold Standard" players.

Advanced SQL Techniques: * Used CTEs and Joins to aggregate data across multiple tables (Matches, Ball-by-Ball, Players).
